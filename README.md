# Приложение «Рейтинг ресторанов» (React)

Интерактивное учебное приложение для просмотра списка заведений с системой оценки.

## Реализованный фунционал:
* **Разделение на компоненты (Components):** интерфейс разбит на части (карточка, рейтинг), чтобы код был чистым и понятным.
* **Работа с состоянием (State):** использовала хук `useState`, чтобы рейтинг менялся на экране сразу после клика.
* **Передача данных (Props):** передача информации от главного файла к отдельным карточкам.
* **Умная отрисовка (Conditional Rendering):** логика, при которой элементы (звездочки рейтинга) меняются в зависимости от данных.
* **Работа со списками:** автоматическое создание карточек из массива данных с помощью метода `map()`.

## Технологии:
* React.js (Функциональные компоненты)
* JavaScript (ES6+)
* CSS3 (Адаптивная верстка)
